Simons-says
Un proyecto creado con vanilla Javascript

Simons-says es un juego en donde el usuario tiene que recordar una secuencia de colores, donde si logra recordar la secuencia y completar los 10 nivel ganara el juego sino perdera y tendra que empezar de nuevo.

Implementacion de Class

La practica de Clases es una muy buena base para el desarrollo de la logica dentro de una website o app, otra experiencia que obtuve durante la creacion de este proyecto fue el contexto de this y sus diferentes facetas dentro de la logica de computo.

Metodos de los Arrays

Los diferentes metodos y propiedades de los Arrays como Array.prototype.fill() o instaciar un new Array me ayudo mucho en la creacion de esta logica, y tambien el metodo .Math() es muy util para generar numeros Random

Condicionales como switch

En este punto de la logica tuve que crear un for loop que me recorriera el array donde se de la secuencia de los colores this.secuencia[i] y pasarlo como paramentro dentro de los switch y asi determinar cual color se iluminaria

Alertas con sweetAlert

En este proyecto aprendi a utilizar esta excelente herraminete que te permite tener una mejor interfaz grafica y mejor experiencia de Usuario
